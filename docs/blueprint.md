# **App Name**: Kadry Online

## Core Features:

- Authentication: Enable admin user login.
- Configuration Management: Provide configuration tabs to manage departments, job titles, managers, nationalities, and clothing lists.
- Active Employee Management: Display a searchable and sortable table of active employees, filterable by department, manager, job title, and nationality.
- Terminated Employee Management: Display a table of terminated employees with additional columns for hire and termination dates.
- New Employee Form: Provide a form to add new employees with validation for all fields.
- Absence Report Generation: Allow users to generate a 'no-show' incident report in A4 PDF format.
- Employee Summary Generation: Use a tool to generate employee summaries based on the current active employee information. Employee information is dynamically updated to generate appropriate summaries.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) for a professional and calm feel.
- Background color: Light gray (#F5F5F5) to ensure readability and reduce eye strain.
- Accent color: A slightly darker blue (#42A5F5) to highlight important interactive elements and buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, machined, objective, neutral look; this font will work for both headlines and body text
- Use simple, modern icons from Material-UI or Ant Design to represent different actions and categories.
- Use a clear, tab-based layout with a side or top navigation menu for easy access to different sections of the application.
- Subtle transitions and animations when navigating between tabs and displaying new data.
